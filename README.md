# Automatismes — 3ème / Brevet

Application web d'entraînement aux automatismes du Brevet des collèges (partie 1).

## Fonctionnalités

- 📝 **S'évaluer** comme au Brevet : 9 questions, 20 min, note /6
- 🎯 **Mon programme** : répétition espacée (max 3 séances/jour)
- ⚔ **Duel** : joueur contre joueur, mini-jeux Puissance 4 / Course de pions
- 📚 **Fiches & outils** : flashcards + exercices de rédaction (glisser-déposer)

91 générateurs couvrant 13 thèmes du programme de 3ème.

## Démarrer

Ouvrir `index.html` dans un navigateur. Aucun serveur nécessaire.

## Mise en ligne

Ce dépôt est configuré pour **GitHub Pages** (voir Settings → Pages).
